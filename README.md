## 📝 Abstract

**Objective:**  
To rigorously evaluate five state‑of‑the‑art optimisers – **Apollo**, **AdamW**, **Lion**, **LAMB**, and **Sophia** – when fine‑tuning a pre‑trained ViT‑B/16 backbone on the CIFAR‑100 image classification task. The study compares accuracy, macro‑F1, ROC‑AUC, Matthews correlation coefficient, training efficiency, and statistical robustness across multiple seeds.

**Methods:**  
All experiments share a fixed hyper‑parameter grid (learning rate, weight decay) tuned per optimiser. Training is repeated over three independent random seeds (42, 123, 777) on the full CIFAR‑100 dataset (50 000 training, 10 000 validation). Mixed‑precision training, cosine annealing with linear warmup, and early stopping are employed. Statistical significance is assessed with paired t‑tests corrected via the Benjamini‑Hochberg FDR procedure; 95% confidence intervals are obtained from 1000‑iteration bootstraps.

**Results:**  
Apollo achieves a competitive accuracy of **0.8737 ± 0.0013** with exceptionally low variance (σ² = 1.66×10⁻⁶). It significantly outperforms Lion (Δ = +0.0462, pFDR = 0.001) and Sophia (Δ = +0.0381, pFDR = 0.005), while remaining on par with LAMB and AdamW. All metrics, learning curves, and LaTeX tables are automatically generated by the provided script.

## 🛠 Implementation Details

| Component               | Specification                                                                 |
|-------------------------|-------------------------------------------------------------------------------|
| **Framework**           | PyTorch 2.0+, torchvision 0.15+                                               |
| **Model**               | ViT‑B/16 (pretrained on ImageNet‑21k)                                         |
| **Input Size**          | 224×224 pixels                                                                |
| **Batch Size**          | 16 (safe for Kaggle T4 ×2)                                                    |
| **Training Epochs**     | 9 (max), early stopping patience = 5                                          |
| **Optimizers**          | Apollo (proposed), AdamW, Lion, LAMB, Sophia (all custom‑written)             |
| **Learning Rates**      | Apollo 3e‑4, AdamW 3e‑4, Lion 1e‑4, LAMB 3e‑4, Sophia 1e‑4                   |
| **Weight Decay**        | Apollo 0.05, AdamW 0.01, Lion 0.01, LAMB 0.05, Sophia 0.01                   |
| **Scheduler**           | Cosine annealing with 2‑epoch linear warmup                                   |
| **Loss**                | Cross‑entropy with Automatic Mixed Precision (AMP)                            |
| **Gradient Clipping**   | Max norm 1.0                                                                  |
| **Reproducibility**     | Seeds 42, 123, 777; deterministic CuDNN (benchmark off)                       |
| **Hardware**            | Kaggle environment (2× Tesla T4 16GB)                                          |
| **Run Time Limit**      | 11.4 hours (hard stop with checkpointing)                                     |
